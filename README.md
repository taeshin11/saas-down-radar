# SaaS Down Radar

Real-Time SaaS Status & Outage Monitor

## About

SaaS Down Radar monitors 20+ major SaaS services and displays their real-time status. Built with vanilla HTML/CSS/JS and Tailwind CSS.

## Features

- Real-time status monitoring for 20+ SaaS services
- Color-coded status indicators (green/yellow/orange/red)
- "Is X Down?" search functionality
- Incident history feed
- Uptime percentage badges
- Auto-refresh every 5 minutes
- Category filters
- Responsive mobile-first design
- i18n support (8 languages)
- Social sharing buttons
- SEO optimized with JSON-LD, Open Graph, sitemap

## Tech Stack

- Frontend: Vanilla HTML/CSS/JS
- Styling: Tailwind CSS (CDN)
- Data: Atlassian Statuspage API
- Hosting: Vercel
- Ads: Google AdSense + Adsterra

## Deployment

```bash
npx vercel --prod --yes
```
